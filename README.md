Gem that split routes.rb into smaller parts.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'routingo'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install routingo

## Usage

  - rails g routingo:install
  - rails g routingo route_name
